Domination Tactical Guide

Master the battlefield with these essential strategies

Endless Domination Mode

  • Endless Survival: Survive as long as possible
  • Growing Map: Map expands 25x25 every 5 minutes
  • Increasing Waves: Bots get harder and more numerous each wave
  • Global Leaderboard: Compete for highest survival time
  • Zoom Controls: Use mouse wheel or +/- keys to zoom
  • Pan Controls: Click and drag to move around the map

Tile Types

  • Blue: Your territory - generates troops
  • Red: Enemy territory - attacks automatically
  • Gray: Neutral land - has defense troops
  • Gold Border: Selected tile ready for action

Combat System

  • Attackers need more troops than defenders
  • Defenders get shield bonus based on difficulty
  • Use bottom slider to select troop amount
  • Only attack adjacent tiles

Strategy for Domination

  • Expand early to increase troop generation
  • Create buffer zones with neutral tiles
  • Consolidate troops before major attacks
  • Watch for bot troop movements on borders
  • Survival is key to high scores

Bots Battle Mode

  • Fixed 8x8 Battlefield: Classic strategy gameplay
  • Elimination: Defeat all bot commanders to win
  • Multiple Opponents: Face 3 AI commanders
  • No Map Growth: Battlefield remains fixed size

1v1 Duel Mode

  • Elite AI: Face a single powerful opponent
  • Tactical Combat: Chess-like strategy on 8x8 grid
  • Single Opponent: Defeat the elite bot commander
  • No Map Growth: Fixed battlefield
